About

What is TREC?

TREC's aim is to increase and support the research community focused on large-scale information evaluation and retrieval by providing a community of users an easy means of comparing their results under a common set of evaluation methods.

What is P_10, P_20 and MAP?
P_10= The relevancy of the first 10 results displayed by the search engine.
P_20= The relevancy of the first 20 results displayed by the search engine.
MAP= MAP stands for Mean Average Precision which is the overall average.

How do I upload a run file?
After a user has registered and signed in to the website, a run file can be uploaded by the "Upload a run file here" button in the user profile page.

What does a track consist of?
A TREC track/task consists of:
A Title , URL to track, year, description of the track, task type, a judgements file (qrels), a list of official measures and notes.
